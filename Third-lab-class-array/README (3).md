# Task 1
Create an array of n integers and fill it with random values. Sort the array in ascending order. Find the sum of all the array elements. Copy the created array to another array. Find the index of the last occurrence of the element.

# Task 2
Create two two-dimensional arrays and fill them with values. Print all elements of the two-dimensional array. Add the two matrices. Transpose the matrix (swap the rows and columns).

# Task 3
Create a jagged array with 3 rows. The first row contains 4 elements, the second row contains 5 elements, and the third row contains 3 elements. Find the sum of all elements in the jagged array. Sort each row of the jagged array.
